## Описание шагов Use Case "Подать заявку":
## Основной сценарий:
 - Пользователь вызывает метод submitApplication контроллера.
 - Контроллер вызывает createApplication у ApplicationService.
 - Сервис получает пользователя по ID через UserService.
 - Внутри UserService происходит валидация данных.
 - Далее из БД извлекается пользователь.
 - Если данные валидны, создается объект заявки и сохраняется в БД через ApplicationRepository.
 - Сервис возвращает заявку в контроллер.
 - Контроллер сообщает пользователю об успешном создании заявки.
